# 🧠 Neurolearn AI

**Neurolearn AI** is an intelligent, dynamic course generation and personalized learning platform. Built originally for the **Bob IBM Hackathon**, this platform leverages the power of advanced AI to instantly curate professional-grade courses, complete with learning paths, assessments, and real-time tutoring.

Whether you want to learn a completely new topic from scratch or upload a specific university syllabus, Neurolearn AI dynamically generates video-based learning materials, quizzes, and career guidance tailored exactly to your needs.

## ✨ Key Features

* **📚 Dynamic Course Generation:** Enter any topic, and the platform will generate a fully structured course like a professional educator, complete with modules, video formats, and reading materials.
* **📄 Syllabus-to-Course Conversion:** Upload your own syllabus document. The AI will analyze it and generate a precise learning path, telling you exactly *what* to learn, *how* to learn it, and *where* to find the resources.
* **🤖 Inbuilt AI Tutor:** Stuck on a concept? The integrated AI tutor (powered by the Gemini API) is available 24/7 to teach, explain, and resolve your doubts in real-time.
* **🛤️ Career Guidance & Learning Paths:** Every generated course includes a strategic learning path and actionable career advice to bridge the gap between learning and employment.
* **📝 Automated Quizzes:** Test your knowledge with dynamically generated quizzes based on the specific material you just covered.
* **📊 Analytics & Progress Tracking:** Includes a secure login system that monitors your daily activity, providing detailed statistics and visual analytics of your learning journey.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js (if applicable)
* Python (if applicable)
* A Gemini API Key

### Installation

1. Clone the repository
```sh
git clone https://github.com/MDAmanMonazir/Neurolearn_AI.git

```


2. Navigate into the project directory
```sh
cd Neurolearn_AI

```


3. Install dependencies
```sh
npm install  # or pip install -r requirements.txt

```


4. Set up your environment variables
* Create a `.env` file in the root directory.
* Add your API keys:
```env
GEMINI_API_KEY=your_api_key_here

```




5. Run the application
```sh
cd neurolearn-AI
npm run dev

```

## 💻 Usage

1. **Sign Up/Login:** Create an account to access the dashboard and enable progress tracking.
2. **Generate a Course:** Use the prompt bar to type a topic you want to learn OR use the **Upload** feature to submit a syllabus.
3. **Learn & Interact:** Follow the generated learning path, watch the curated video formats, and chat with the AI tutor if you need deeper explanations.
4. **Take Quizzes:** Validate your learning at the end of each module.
5. **Track Progress:** Check the analytics dashboard to monitor your daily streaks and topic completion rates.
